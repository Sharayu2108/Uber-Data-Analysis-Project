# Transportation-Performance-Analysis-Uber-
This project analyzes Uber ride booking data to derive insights into ride patterns, cancellations, customer behavior, revenue, and ratings. It combines Excel dataset analysis and a Power BI dashboard to simulate a real-world Data Analyst project.
